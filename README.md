PROVA DE SISTEMA DiSTRIBUIDOS

 Sistema Distribuído Cliente/Servidor em Python

Este projeto foi desenvolvido como atividade da disciplina de Sistemas Distribuídos, com o objetivo de implementar uma arquitetura distribuída do tipo cliente-servidor utilizando Python, sockets e threads.

---

 Objetivo do Projeto

Desenvolver um sistema distribuído baseado no modelo Produtor/Consumidor, onde:

- Produtores criam requisições;
- Um servidor produtor armazena as requisições em uma fila compartilhada;
- Consumidores solicitam requisições;
- Um servidor consumidor intermedeia a comunicação entre consumidores e o servidor produtor.

O sistema demonstra conceitos fundamentais de Sistemas Distribuídos, como:

- Comunicação Cliente/Servidor;
- Comunicação entre servidores;
- Concorrência;
- Buffer compartilhado;
- Sincronização;
- Processamento paralelo;
- Uso de Threads.
